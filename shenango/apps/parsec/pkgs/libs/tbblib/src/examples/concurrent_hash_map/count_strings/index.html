To Build

General build directions can be found here.

Usage

count_strings [verbose] [N] [P]
N is the number of strings and P is the number of threads to be used. The "verbose" option enables printing of extra information during execution.
To run a short version of this example, e.g., for use with Intel® Threading Tools:
Build a debug version of the example (see the build directions).
Run it with a small number of strings and the desired number of threads, e.g., count_strings 10000 2.
