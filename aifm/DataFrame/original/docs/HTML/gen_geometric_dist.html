template<typename T>
std::vector<T>
gen_geometric_dist(std::size_t n, const RandGenParams<T> &params = { }); 		Produces random non-negative integer values i, distributed according to
discrete probability function:
          P(i|p)=p⋅(1−p)i
It returns the vector of results
Optional parameters to set: prob_true, seed 		T: Type can only be [unsigned] char, [unsigned] short, [unsigned] int, [unsigned] long int, or [unsigned] long long int
n: Number of numerics to generate
params: List of all applicable parameters, see DataFrameTypes.h
